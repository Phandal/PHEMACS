* Overview
To start, I was a vim/neovim user for about 3 years. After reading a lot about Emacs, I have decided to make the switch and give Emacs and actual shot. By no means is this perfect as I'm still quite new to Emacs and I don't know everything (But do you ever really know everything there is to know about Emacs?). My main focus is trying to make this config work for both my work and personal PC. My work computer uses emacs for windows and my personal computer uses emacs compiled with native compilation. I spent to much time trying to manage 2 or 3 different repos for work vs personal configs and wanted to make them all work off of the same one!

* Installation
*I haven't wrote the install script yet*
** Linux
1. Clone this repo
#+begin_src bash
git clone https://github.com/Phandal/PHEMACS.git
#+end_src
2. Run install.sh
#+begin_src bash
cd PHEMACS
./install.sh
#+end_src
** Windows
1. Clone this repo
#+begin_src batch
git clone https://github.com/Phandal/PHEMACS.git
#+end_src
2. Run install.sh
#+begin_src batch
cd PHEMACS
./install.sh
#+end_src
3. Set the sytem environment variable ~$HOME~ to whatever ~%USERPROFILE%~ evaluates to. For example:
#+begin_src batch
echo %USERPROFILE%
-> C:\Users\Phandal
#+end_src
